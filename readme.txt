Facebook Pixel Pal inserts your Facebook Pixel Code into it's proper location.

== Description ==

Easily add your Facebook Pixel Code to your WordPress website. 

== Installation ==

1. Upload plugin files to your plugins folder, or install using WordPress built-in Add New Plugin installer.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. From the Dashboard go to Settings>Facebook Pixel Pal.
4. Enter/paste your Facebook Pixel Code.
5. Click the "Save Changes" button and you are done.
